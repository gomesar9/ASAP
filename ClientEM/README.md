# Event Monitor Client

**Before run ClientEM make sure that EventMonitor10 driver is running**.  
Recomended use:  
run IPython(3) in `ClientEM/src/emclient/`

```
[1] %run emc_wrapper.py
[EMC]> help
[EMC]> cfg
...
[EMC]> start
[EMC]> stop
[EMC]> exit
```

`cfg` enables you to change all configurations
